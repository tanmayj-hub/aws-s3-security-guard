# Security Policy

## Supported Versions
This project is a learning/portfolio repo. Only the latest version on the `main` branch is supported.

## Reporting a Vulnerability
If you discover a security issue:
- **Do not** open a public GitHub issue with sensitive details.
- Instead, message me on LinkedIn or open a GitHub issue with **high-level context only** and ask for a private channel to share details.

## Scope
In scope:
- Vulnerabilities in the scanner/remediation logic
- CI/CD workflow issues (GitHub Actions)
- IAM/OIDC replication guidance problems that could cause unsafe usage

Out of scope:
- AWS account misconfigurations outside the scope of this repo
- Third-party AWS service vulnerabilities
